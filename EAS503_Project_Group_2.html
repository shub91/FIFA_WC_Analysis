Project Description and impact on community¶

  • Description - Analysis of in-match football data to:
    1. Glean out non-intuitive insights
    2. present highlights and lowlights of the tournament
    3. Come up with intereseting metrics to determine team's and player's performance
  • Why? - Most of the analysis that you see on the internet gives us a superficial view of what happened in the tournament therefore leaving many unusual trends hidden and questions unanswered. This analysis goes beyond such works and unlocks the hidden trends in order to help us understand the strategies and their impact.
  • Impact on sports and entertainment community:
    1. Helps football clubs make decisions based on the world cup performance of players.
    2. If built on a more comprehensive data set it can be used by teams to devise strategies based on the opponents.
    3. Can help football clubs save big on scouts and pick out players who aren't visible to naked eyes.
  • The Data - The data has been taken from 'Statsbomb.com' - a football analytics firm run by Ted Knutson. Data is a stream of events occuring at second level for any given match
    Link: https://github.com/statsbomb/open-data/tree/master/data

All our analysis has been compiled in a Dashboard using Plotly. Link: https://plot.ly/dashboard/asahay:16/view

var fs = require("fs");

var ls = fs.readdirSync("./");

var files = ls.filter(function(filename) {
    return filename.match(/^[0-9]+.json/) !== null;
});

var full = [];

var data = null;
